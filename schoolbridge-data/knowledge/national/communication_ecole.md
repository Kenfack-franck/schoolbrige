# Communiquer avec l'École

## Les canaux de communication

### Email
Le moyen le plus courant pour contacter un enseignant. Le ton doit être formel (Sehr geehrte Frau/Herr..., mit freundlichen Grüßen). La réponse peut prendre 1 à 3 jours — les enseignants ont beaucoup de sollicitations.

### Carnet de correspondance / Mitteilungsheft
Beaucoup d'écoles primaires utilisent un cahier où parents et enseignants échangent des messages courts. Vérifier le cahier tous les jours.

### Applications scolaires
Certaines écoles utilisent des apps (SchoolFox, Sdui, Untis) pour communiquer. Installer l'app recommandée par l'école.

### Téléphone
Pour les urgences uniquement. Appeler le secrétariat, pas directement l'enseignant.

## Situations courantes et comment réagir

### Mon enfant est malade
1. Appeler l'école le matin même (avant 8h si possible)
2. Envoyer une excuse écrite dans les 3 jours (Entschuldigung)
3. Modèle : "Hiermit entschuldige ich das Fehlen meines Sohnes/meiner Tochter [Prénom Nom], Klasse [X], am [Date] wegen Krankheit. Mit freundlichen Grüßen, [Votre nom]"
4. Au-delà de 3 jours d'absence, un certificat médical (Attest) est généralement demandé

### Je veux un rendez-vous avec l'enseignant
1. Envoyer un email : "Sehr geehrte/r Frau/Herr [Nom], ich bin die Mutter/der Vater von [Prénom], Klasse [X]. Ich würde gerne einen Termin mit Ihnen vereinbaren, um über die schulischen Leistungen meines Kindes zu sprechen. Mit freundlichen Grüßen, [Votre nom]"
2. Ou utiliser les heures de permanence (Sprechstunden) affichées sur le site de l'école

### Mon enfant a un problème avec un autre élève
1. D'abord en parler avec l'enseignant principal (Klassenlehrer)
2. Si le problème persiste, contacter le directeur
3. En cas de harcèlement (Mobbing), l'école a l'obligation d'agir
4. Un médiateur scolaire (Schulsozialarbeiter) peut intervenir si l'école en a un

### Je ne comprends pas le bulletin de mon enfant
1. Demander un rendez-vous avec le Klassenlehrer pour qu'il explique les notes
2. Les notes vont de 1 (très bien) à 6 (très insuffisant) — 4 est la note minimale pour passer
3. Les appréciations (Bemerkungen) donnent des indications sur le comportement et la participation

### Je veux contester une décision
1. D'abord : entretien avec l'enseignant concerné
2. Ensuite : entretien avec le directeur (Schulleiter)
3. Si pas de résolution : recours écrit auprès du Schulamt (autorité scolaire)
4. Le Elternbeirat peut aussi être un allié dans certaines situations
